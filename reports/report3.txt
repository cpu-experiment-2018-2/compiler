
* 問1
FL実験の第一回の6問のfilter_frで使っていた.
let filter_fr f l =
  let func x acc = if f x then x :: acc else acc in
  fold_right func l []
これを自由変数が無いようにすると
llet filter_fr f l =
  let func x (f,acc) = if f x then x :: acc else (f,acc) in
  fold_right func l (f,[])
となり非常にめんどくさい
